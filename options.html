Criar Utilizadores

Após instalar a extensão deve abrir a página do Facebook e iniciar a sua sessão.
Caso já tenha a página do Facebook aberta antes de instalar a extensão deve obrigatoriamente recarregar a página referida.
Posto isto, pode carregar no botão Novo Utilizador e seguir as instruções.
É importante recordar que na eventualidade de não se recordar do código que definiu para a sua conta, não há maneira de o recuperar. Caso isto aconteça, terá de desisntalar e voltar a instalar esta extensão para poder criar uma nova conta de utilizador.

Atenção: Após criar a conta deve recarregar a página do Facebook para as alterações fazerem efeito.

Desativar Utilizadores

Existe a opção de desativar uma conta sem ter de a remover.
Desta forma todas as mudanças na sua página do Facebook serão desativadas e o estilo oficial irá aparecer de novo. Para efetuar esta operação, só tem de carregar no botão Desativar Conta referente ao seu utilizador e introduzir o seu código.
Para voltar a ativar a conta só tem de efetuar o processo inverso e todas as suas definições voltam a aparecer na página do Facebook.

Atenção: Após qualquer operação deve recarregar a página do Facebook para as alterações fazerem efeito.

Editar e Remover

O utilizador tem a opção de editar o seu código. Para tal basta pressionar o botão Editar Código e irá aparecer uma janela em que o utilizador terá de introduzir o código atual e o novo código pretendido.
Se assim o desejar também pode remover a sua conta. Pressione o botão Remover Utilizador, introduza o seu código e confirme se deseja eliminar a conta.

Atenção: Após qualquer operação deve recarregar a página do Facebook para as alterações fazerem efeito.

Personalizar

Para personalizar a página do Facebook a seu gosto, é permitido efetuar algumas mudanças.
Antes de começar certifique-se que tem a sua sessão iniciada na página do Facebook e que tem uma conta criada nesta aplicação.

Para personalizar a barra de topo do Facebook o utilizador pode escolher uma cor a seu gosto ou introduzir um link para uma imagem. Se escolher por inserir uma imagem deve carregar no botão Guardar imagem atual após introduzir um link.

Pode também escolher uma cor ou uma imagem para o fundo do Facebook da mesma maneira na secção Fundo.

Também pode escolher um novo tipo de letra e tamanho para a sua página na secção Letra. Para tal deve escolher uma das opções fornecidas no tipo de letra e um tamanho. Após isto pressione o botão Guardar tipo de letra.

Para cada uma destas mudanças tem o respectivo botão para as remover, devolvendo o aspeto original à sua página do Facebook.

Atenção: Após qualquer operação deve recarregar a página do Facebook para as alterações fazerem efeito.
